A - Number of Multiples
シンプルな全探索で終わり。O(1) でもできると思う。 

aising2020_a.py

l,r,d = map(int,input().split())
ans = 0
for i in range(l,r+1):
    if i%d == 0:
        ans += 1
print (ans)
