AtCoder Judge System Update Test Contest 202004

Judge System Update Test Contest 202004 に参加しました。

A

A - Walking Takahashi
現在位置がLとRの間ならそのまま出力、Lより小さいならLを、Rより大きいならRを出力すれば良い

提出
ちょっと冗長だった。absは不要だな。
以下でOK

#!/usr/bin/env python3
# -*- coding: utf-8 -*-
S,L,R = map(int,input().split())
if S < L:
    print (L)
elif R < S:
    print (R)
else:
    print (S)

こっちのほうがシンプル。

B

B - Picking Balls
RとBの配列を持って置き、入力を受けてR/Bどちらかの配列に append して、入力を受け終わったらソートして R → B の順番に出力すればよい。
